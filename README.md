# 🚀 Journi.dev: Your Interactive Coding Journey

Welcome to **Journi.dev**! 

Are you feeling lost in the endless sea of programming tutorials? Not sure what to learn next to land that Backend or Frontend role? **Journi.dev** is here to guide you. We transform static learning paths into an interactive, gamified, and community-driven experience powered by AI.

## ✨ What is Journi.dev?

Journi.dev is an Interactive Learning Management System (LMS) designed specifically for developers. Instead of just giving you a list of topics to read, we provide a **Skill Tree** where you must complete practical, hands-on coding challenges to unlock your next steps. 

## 🔥 Key Features

* 🗺️ **Interactive Skill Trees:** Visualize your learning journey. Complete prerequisite "Nodes" to unlock advanced technologies.
* 🤖 **AI-Powered Code Review:** Say goodbye to boring multiple-choice tests. Submit your actual code via GitHub! Our AI Agent will pull your repo, perform static analysis, and give you professional feedback on your Clean Code, Design Patterns, and Security.
* 🤝 **Milestone Communities:** Don't learn alone. Connect with developers who are at the exact same stage as you in our Cluster-based discussion rooms.
* ⭐ **Peer-Review & Leaderboards:** Rank up by helping others! Review code from junior developers, maintain your daily learning streak, and climb the specialized leaderboards.

## 🛠️ Built With

* **Frontend:** React.js, TypeScript, Tailwind CSS, React Flow (Vercel)
* **Backend:** Java Spring Boot (AWS EC2)
* **Database:** PostgreSQL (Primary), Redis (In-memory/Caching)
* **AI & Integrations:** GitHub API / Webhooks, Gemini API
* **DevOps:** Docker
