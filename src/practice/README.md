# Journi.dev practice contracts

This directory contains learner-visible starter contracts for deterministic GitHub practice. Work in a fork or copy the relevant directory into a public repository, keep the documented public APIs stable, and submit the exact 40-character commit SHA.

Public starter repositories linked from seeded `PRACTICE` and `PROJECT` nodes are tracked in `docs/PRACTICE_STARTER_REPOSITORIES.md`. Keep that catalog, the seed dataset, and the challenge `expectedArtifacts` aligned whenever a starter contract changes.

Only `collections-and-generics` is enabled as the initial grading pilot. The remaining roadmap briefs stay visible but disabled until their isolated grader contracts and abuse checks are validated.
