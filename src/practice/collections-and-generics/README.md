# Collections and Generics practice

Implement `LibraryCatalog` without raw collection types or unchecked casts.

The catalog must:

1. Store each ISBN at most once.
2. Return a book by ISBN using `Optional`.
3. Group books by author.
4. Return deterministic, read-only results from `groupByAuthor`.

Do not rename the package, classes, record components, constructor, or public methods. The evaluator compiles these source files with server-owned tests.
